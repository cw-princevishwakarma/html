This is a paragraph of text to practice HTML structure. This is an emphasized text using the em tag, and it converts this to italic visually. This is a strong text using the strong tag, and it converts this to bold visually

If we have b and i tags, why would we use strong and em? Because screen readers (software) read these with different tones: em with calmness and strong with stress.
